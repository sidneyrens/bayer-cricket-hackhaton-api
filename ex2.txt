1. `git checkout exercise2`
2. in the AWS console go to DynamoDB create a table called `user` with `_id` as the key
3. add two records to user 
{
  "_id": "rommel",
  "file": "rommel.docx" //any file would do
} and
{
  "_id": "yourname",
  "file": "yourname.docx"
}
4. create those two files in word if you don't have them
5. go to S3, create a bucket called 'docs-serverless-class` in US East (N. Virginia) and upload those two files
6. do npm install and run locally
